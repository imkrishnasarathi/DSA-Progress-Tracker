## Problem statement - Create flowchart and pseudocode for the following:

### 1. Input a year and find whether it is a leap year or not.

#### Pseudocode

```
start
input year

if year%100 != 0 and year%4 == 0:
  output "leap year"
else if year%400 == 0:
  output "leap year"
else:
  output "not a leap year"

exit
```
